- Year: 2024

### One sentence takeaway
  - In this study, we quantitatively analyze 1,756 AI-written Python code snippets in the DevGPT dataset and evaluate them for quality and security issues.

### Most intersting find
  - Concerning security issues, there are no significant differences between ChatGPT-generated and ChatGPT-modified code.


## Methodology
  - Two research questions:
    - RQ1: How is the quality of Python code developed with assistance from ChatGPT?
    - RQ2: How prevalent are different security vulnerabilities in ChatGPT-aided Python code?
  - We perform a comparative analyses between ChatGPT-modifed and ChatGPT-generated code.
  - Pylint, we separately analyze the ChatGPT-modifed and ChatGPT-generated code snippets to capture key code quality issues of four types: errors(E), conventions (C), warnings (W), and refactoring (R).
  - Using Bandit, we again separately analyze the ChatGPT-modifed and ChatGPT-generated code snippets for capturing security vulnerabilities in them.

#### Analysis:
  - code quality: Pylint 3.0.2
  - security: Bandit 1.7.5

#### CWE classification:
  - Yes provided by Bandit


## Experimental Setup

#### LLM model(s) used:
  - gpt-3.5-turbo (only used for extracting code snippets from prompts in dataset)

#### Programming languages used:
  - Python

#### Prompting strategy:
  - No prompting

#### Dataset / tasks:
  - 1,756 AI-written Python code snippets in the DevGPT dataset (total: 17,622 conversations, 12,031 snippets).
  - With gpt-3.5-turbo we identify prompts from the conversations that contain Python code in the responses and we extract any Python code snippets contained within these prompts.
  - Distinguish the code snippets as either generated by ChatGPT from scratch (ChatGPT-generated) or modified user-provided code (ChatGPT-modified)
  - Using cosine similarity, we distinguish 213 python snippets as ChatGPT-modified. The rest 1,543 are regarded as ChatGPT-generated.

#### Evaluation metric
  - four generic metrics applied for both quality and security analyses


## Results
  - The results reveal that ChatGPT-modified code more frequently displays quality issues compared to ChatGPT-generated code.
  - Concerning security issues, there are no significant differences between ChatGPT-generated and ChatGPT-modified code.

### Types of vulnerabilities identified
  - hardcoded secrets
